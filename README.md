# 浙江高考化学题库（网页版）

浏览器里打开的刷题页面：方程式 + 知识点，支持**选择题 / 判断题**、错题本、统计、打卡。数据保存在本机浏览器。

## 本地运行

```bash
cd chem-equation-quiz
npm install
npm run dev
```

浏览器打开终端里提示的地址（一般为 `http://127.0.0.1:5173/`）。

### 用手机浏览器打开（与电脑同一 WiFi）

1. 电脑上执行 **`npm run dev:lan`**（不要用普通的 `dev`，这样才能被手机访问）。
2. 终端里会显示 **Network** 地址，形如 `http://192.168.x.x:5173/`。
3. 手机连**同一个 WiFi**，在浏览器里输入这个地址即可。
4. 若打不开：检查电脑防火墙是否放行 **5173** 端口。

不在同一网络时，请使用下面「打包部署」把网站发布到公网，再用手机访问你的域名。

## 打包成静态网页

```bash
npm run build
```

生成目录 `dist/`，可部署到任意静态托管（GitHub Pages、Vercel、阿里云 OSS、学校服务器 Nginx 等）。

本地预览打包结果：

```bash
npm run preview
```

## 题库修改

编辑 `src/data/equations.ts` 与 `src/data/facts.ts`，保存后开发环境会自动刷新。

## 说明

- 本项目为 **纯前端网页**，无需微信、无需后端。
- **Cursor 内置 Simple Browser 有时会长期缓存旧页面**（仍显示深色、手写输入框）。请用 **系统自带 Chrome / Safari** 打开同一地址，或按 **Cmd+Shift+R** 强制刷新。
- 正确界面应为：**白底**、顶部「浙江高考化学题库」、题目区有 **选择题 / 判断题** 按钮与 **四个选项**。
