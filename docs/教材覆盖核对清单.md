# 浙江高考化学教材覆盖核对清单（人教版）

说明：
- 本清单用于“逐章核对是否已入库（方程式 + 记忆点）”。
- 当前系统中题库来源文件为 `src/data/equations.ts` 与 `src/data/facts.ts`。
- 勾选规则建议：每小节至少有核心方程式/核心概念题后再勾选。

## 必修一
- [ ] 第一章 物质及其变化（物质分类、离子反应、氧化还原）
- [ ] 第二章 海水中的重要元素（钠、氯）
- [ ] 第三章 铁 金属材料（铁及其化合物、铝及其化合物）
- [ ] 第四章 物质结构 元素周期律（若教材版本将部分内容并入必修二，可跳过）
- [ ] 氮、硫及其化合物专题（教材编排名称可能略有差异）

## 必修二
- [ ] 第一章 物质结构与性质（化学键、周期律复习与拓展）
- [ ] 第二章 化学反应与能量（反应热、速率与平衡基础）
- [ ] 第三章 有机化合物基础（烷烃、烯烃、芳香烃、乙醇/乙酸）
- [ ] 第四章 化学与可持续发展（可转化为知识点记忆题）

## 选必一 化学反应原理
- [ ] 第一章 化学反应速率
- [ ] 第二章 化学平衡
- [ ] 第三章 水溶液中的离子反应与平衡（电离、水解、沉淀溶解）
- [ ] 第四章 电化学基础（原电池、电解池、金属腐蚀与防护）

## 选必二 物质结构与性质
- [ ] 第一章 原子结构与元素性质
- [ ] 第二章 分子结构与性质（价层电子对互斥、杂化、极性、氢键）
- [ ] 第三章 晶体结构与性质（晶胞、晶体类型、性质比较）
- [ ] 过渡元素与配位化学基础（若教材包含）

## 选必三 有机化学基础
- [ ] 第一章 有机化合物结构与同分异构
- [ ] 第二章 烃和卤代烃
- [ ] 第三章 烃的含氧衍生物（醇、醛、羧酸、酯）
- [ ] 第四章 生物大分子与高分子材料（糖类、蛋白质、合成高分子）
- [ ] 有机合成与推断专题（常见路线、官能团转化）

## 下一步补库建议
- [ ] 逐章补齐“教材例题/实验探究”中的关键方程式
- [ ] 每章至少整理 20~40 条记忆型题（定义、规律、现象、实验结论）
- [ ] 统一同义答案（例如“V形/折线形”）以提升判题容错
- [ ] 为每题补充 `chapter` 与更细粒度 `tags`，便于按章练习
